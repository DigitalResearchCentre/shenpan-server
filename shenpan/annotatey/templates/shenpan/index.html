Welcome to the Shenpan home page

Shenpan is a microservice server for making and documenting decisions when regularizing collations or enhancing document searches. It is currenty in pre-alpha.

This page concerns the software that you can download and run on your own server. If you just want to use our web tool, visit annotation.bham.

Shenpan was started during the Interedtion Leuven bootcamp 2011.
